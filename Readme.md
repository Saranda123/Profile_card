# 🚀 ProfileCard

A modern and responsive **Profile Card** front-end project built with **Vite**, **React**, **JavaScript**, and **Sass**.  
This project showcases a clean UI component designed to display user profile information in an elegant and reusable way.

---

## 📌 Overview

ProfileCard is a simple yet visually appealing front-end project focused on:

- Component-based architecture with React
- Fast development environment powered by Vite
- Modular and maintainable styling using Sass
- Responsive design for different screen sizes

The goal of this project is to practice modern front-end development workflows and create reusable UI components.

---

## 🛠 Tech Stack

- ⚡ **Vite** – Fast build tool and development server  
- ⚛ **React** – JavaScript library for building user interfaces  
- 🎨 **Sass (SCSS)** – CSS preprocessor for better styling structure  
- 🟨 **JavaScript (ES6+)** – Modern JavaScript features  

---

## ✨ Features

- Responsive profile card layout  
- Clean and modern design  
- Reusable React component structure  
- Organized Sass folder structure  
- Easy customization for different users

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/profilecard.git
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run development server

```bash
npm run dev
```

### 4️⃣ Build for production

```bash
npm run build
```
